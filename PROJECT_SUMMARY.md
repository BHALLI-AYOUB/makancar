### 🎨 Design System

**Colors:**
- Primary: Gold (#fbbf24 - Amber-400)
- Secondary: Dark (#000 - Black)
- Accents: Green, Blue, Purple, Red (for specific actions)

**Typography:**
- Headings: Playfair Display (serif)
- Body: Inter (sans)
- Font Sizes: Scale from 12px to 72px

**Spacing:**
- Base Unit: 4px (Tailwind)
- Sections: 96px vertical spacing (py-24)
- Components: 16px-24px padding

**Animations:**
- Duration: 300ms - 800ms
- Easing: Default (ease-out)
- Triggers: Scroll, hover, click
